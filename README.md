# Курс Профессия Data Engineer с нуля до junior

## Описание проекта

Курс предлагает комплексное освоение необходимого инструментального стека: 
от фундаментальных основ SQL, Python, Linux и GIT до продвинутых технологий 
обработки больших данных, таких как Hadoop, Spark, Kafka, Airflow и ClickHouse.
